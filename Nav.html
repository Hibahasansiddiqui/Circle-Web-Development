Semantic HTML Elements

Navigation Tag

The nav element defines a set of navigation links. This can be found in any website, without a menu or navigation bar, websites will surely be like a sailor sailing their boat without a map or compass.

Section Tag

Section tag defines the section of documents such as chapters, headers, footers or any other sections. The section tag divides the content into section and subsections. The section tag is used when requirements of two headers or footers or any other section of documents needed.

Footer Tag

The footer element is a structural element used to identify the footer of a page, document, article, or section. A footer typically contains copyright and authorship information or navigational elements pertaining to the contents of the parent element.
